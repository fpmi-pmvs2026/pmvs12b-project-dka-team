# Главная страница

Документация проекта **NewsAppDemo** для Wiki/GitHub Pages.

## Краткое описание задачи

Необходимо реализовать Android-приложение для получения новостей через NewsAPI с возможностью:
- управлять подписками на темы;
- просматривать и обновлять новости;
- настраивать язык, интервал обновления, уведомления и режим Wi-Fi only;
- хранить данные локально и выполнять фоновое обновление.
