# Схема базы данных

- [Скачать/открыть SQL-схему](db/schema.sql)

## Краткое описание

- Таблица `subscriptions` хранит список тем подписок.
- Таблица `articles` хранит статьи, связанные с конкретной темой подписки.
- Связь между таблицами: `subscriptions (1) -> (N) articles`.
- При удалении подписки связанные статьи удаляются каскадно (`ON DELETE CASCADE`).
