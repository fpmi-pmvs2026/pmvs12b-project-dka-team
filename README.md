# Project Name

**NewsAppDemo (pmvs12b-project-dka-team)**

## Description

NewsAppDemo — учебное Android-приложение для получения новостей через [NewsAPI](https://newsapi.org/).  
Пользователь может подписываться на интересующие темы, просматривать список статей и открывать полные материалы в браузере.  
Приложение поддерживает ручное и фоновое обновление данных, а также уведомления о новых публикациях.  
В настройках доступны выбор языка, управление интервалом обновления и режим обновления только по Wi-Fi.  
Проект реализован на Kotlin с использованием Jetpack Compose, Clean Architecture и современного Android-стека.

## Installation

1. Клонируйте репозиторий вместе с сабмодулями:

```bash
git clone --recurse-submodules <URL_ЭТОГО_РЕПОЗИТОРИЯ>
cd pmvs12b-project-dka-team
```

2. Если репозиторий уже был клонирован без сабмодулей, инициализируйте их:

```bash
git submodule update --init --recursive
```

3. Убедитесь, что установлены:
- Android Studio (актуальная версия);
- JDK 11;
- Android SDK (`compileSdk = 36`, `minSdk = 24`, `targetSdk = 36`).

4. Получите API-ключ на [NewsAPI](https://newsapi.org/) и создайте в папке `code` файл `keystore.properties`:

```properties
NEWS_API_KEY="YOUR_API_KEY"
```

5. Откройте папку `code` в Android Studio, дождитесь синхронизации Gradle и запустите конфигурацию `app` на эмуляторе или устройстве.

## Usage

- После первого запуска выберите интересующие темы и активируйте подписки.
- На главном экране просматривайте карточки новостей: источник, дату, описание и изображение.
- Открывайте полную статью в браузере или делитесь ссылкой через системный Share.
- В разделе настроек можно изменить язык поиска, интервал фонового обновления и параметры уведомлений.
- Для проверки сборки и тестов из директории `code` используйте:

```bash
./gradlew assembleDebug
./gradlew test
./gradlew connectedAndroidTest
```
